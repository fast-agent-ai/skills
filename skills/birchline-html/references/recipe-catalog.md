# Birchline Recipe Catalog

Use this catalog to choose among the packaged Birchline artifact recipes. Every
recipe named here is bundled with the skill and plugin.

## Packaged Recipes

| Recipe | Use when |
| --- | --- |
| `module-explainer.md` | Explain a module, subsystem, request path, runtime flow, or part of a codebase. |
| `process-explainer.md` | Explain a workflow, lifecycle, handoff, operational process, or team procedure. |
| `implementation-plan.md` | Propose a build/change plan with milestones, interface contracts, risks, and verification. |
| `design-directions.md` | Compare technical, product, workflow, API, or visual directions and recommend a path. |
| `code-review.md` | Present findings-first review risks, affected files, evidence, and reviewer checklist. |
| `pr-change-writeup.md` | Explain a change, branch, migration, or pull request for reviewers and stakeholders. |
| `status-or-incident-report.md` | Summarize status, an incident, an investigation, recovery, or follow-up plan. |
| `slide-deck.md` | Create a compact briefing deck, weekly update, decision readout, or presentation narrative. |
| `flow-diagram.md` | Draw and explain a technical flow, lifecycle, state machine, deploy path, or decision tree. |
| `numeric-data.md` | Summarize arbitrary metrics, benchmarks, dashboards, comparisons, and plotted data. |

## Reusable Patterns

- Use `birchline-patterns.md` for core layout and component guidance.
- Use `numeric-data.md` for chart/table treatment even inside other artifact
  types.
- Use `flow-diagram.md` when a module or process explainer needs a large
  annotated diagram rather than a small inline flow.
- Use `slide-deck.md` when the output is meant to be presented, scanned slide by
  slide, or printed as a briefing.

## Remaining Gaps

These shapes are not yet first-class packaged templates:

- Animation or interaction prototype writeups.
- Triage/configuration/editor dashboards.
- Dedicated research/learning concept explainers.

When one of these is requested, choose the nearest packaged recipe and make the
intended structure explicit in the renderer brief.
